# Streamflix

Streamflix é um sistema de gerenciamento de catálogo de filmes que permite aos usuários adicionar, remover, editar e visualizar filmes. O sistema também oferece funcionalidades para registrar interações dos usuários e recomendar filmes com base no histórico de visualização.

## Funcionalidades

- **Gerenciamento de Catálogo**: Adicione, remova e edite filmes no catálogo.
- **Exibição de Filmes**: Visualize todos os filmes disponíveis no catálogo.
- **Registro de Interações**: Registre interações dos usuários com os filmes assistidos.
- **Recomendações**: Receba recomendações de filmes com base no histórico de visualização.
- **Salvar e Carregar Dados**: Salve o catálogo e o histórico de usuários em arquivos CSV.

## Tecnologias Utilizadas

- Linguagem de Programação: C
- Bibliotecas: Padrões da biblioteca C (stdio.h, stdlib.h, string.h)
